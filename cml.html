                In your `word_guess_game` app, define the models for the database. You can use Django's built-in models to represent Dutch words and their attributes.
                
                ```python
                # models.py
                from django.db import models
                
                class DutchWord(models.Model):
                    DUTCH_WORD_TYPES = [
                        ('noun', 'Noun'),
                        ('verb', 'Verb'),
                        ('adverb', 'Adverb'),
                    ]
                
                    dutch = models.CharField(max_length=100, unique=True)
                    word_type = models.CharField(max_length=10, choices=DUTCH_WORD_TYPES)
                    example = models.TextField()
                    translations = models.JSONField()
                    dutch_antonym = models.CharField(max_length=100, blank=True)
                
                    def __str__(self):
                        return self.dutch
                ```
